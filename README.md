<dir>
  <h1> 🧩 About the Game </h1>
  <h3>
    Gobblet Gobblers is a strategy game similar to tic-tac-toe, but with a twist:
    players can place larger pieces on top of smaller ones (“gobbling” them) to gain advantage.
    <br>This project includes:<br><br>
    - Pixel-art game board and animations<br>
    - Drag-and-drop piece interaction<br>
    - Bot opponent<br>
    - Win-detection logic<br>
    - Basic game states (menu, playing, victory screen)<br>
  </h3>
